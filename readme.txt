== Description ==

Reminder Mail plugin Is used to send mails to clients or others from admin panel to remind something.

== Support =

You can find (http://jobsmug.com/reminder-mail-system/).

== Installation ==

1. Upload the entire `reminder-mail-system` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Reminder Mail' menu in your WordPress admin panel.
you can send mails to any user from there.

You can find 5 text boxes there.
1. From Email
   It will take admin email Id for wordpess site automatically. and you can change it also.
2. From Name
   It will come from blog, you can change it later.
3. To email
   Email Id of other persons to whom you want to send a mail.
4. Subject
   The subject of the mail.
5. Message
   you can enter message.
